STEP 1 → REST Controller (Spring Boot API)
@RestController
@RequestMapping("/employ")
public class EmployRestController {

    @GetMapping
    public List<Employ> showEmploy() {
        return List.of(
                new Employ(1, "Prasanna", "IT", 45000),
                new Employ(2, "Rahul", "HR", 35000)
        );
    }
}

STEP 2 → MVC Controller to Call REST
@Controller
public class EmployMvcController {

    @Autowired
    private RestTemplate restTemplate;

    @Bean
    public RestTemplate restTemplate() {
        return new RestTemplate();
    }

    @GetMapping("/showEmploy")
    public String showEmploy(Model model) {
        String url = "http://localhost:8080/employ";

        Employ[] list = restTemplate.getForObject(url, Employ[].class);

        model.addAttribute("employList", list);

        return "showEmploy";   // showEmploy.jsp
    }
}

STEP 3 → JSP Page (showEmploy.jsp)

Add JSTL dependency (if not already):

<dependency>
    <groupId>javax.servlet</groupId>
    <artifactId>jstl</artifactId>
</dependency>

showEmploy.jsp
